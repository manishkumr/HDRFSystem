Sectionizer Version 1.0.0 can be run on windows,linux or OSX.

Note:
1. Input File/Files should be in i2b2 2014 format only


Pre-requisites:
1. Java jdk 1.8 or higher

Steps:
1. Unzip Sectionizer1.0.0.zip to directory hereafter refered as SECTIONIZER_HOME
2. Open file config.properties located under SECTIONIZER_HOME using a text editor of your choice.
3. Change configuration parameter as per the requirements.
4. Open a terminal and navigate to SECTIONIZER_HOME
5. Execute command " java -jar Sectionizer1.0.0.jar 
7. Once system runs successfully output file will written under user specified directory.
