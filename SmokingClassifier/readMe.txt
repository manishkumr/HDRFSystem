SmokingClassifier Version 1.0.0 can be run on windows,linux or OSX.

Pre-requisites:
1. Java jdk 1.8 or higher

Steps:
1. Unzip SmokingClassifier1.0.0.7z to directory hereafter refered as SmokingClassifier_HOME
2. Open file config.properties located under SmokingClassifier_HOME using a text editor of your choice.
3. Change configuration parameter as per the requirements.
4. Open a terminal and navigate to SmokingClassifier_HOME
5. Execute command " java -jar SmokingClassifier1.0.0.jar 
7. Once system runs successfully output file will written under user specified directory.
